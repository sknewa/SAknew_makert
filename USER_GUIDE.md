## Getting Started

### Creating an Account
1. Open the Saknew Market app
2. Tap "Register" on the login screen
3. Enter your email,  and password, 
4. Tap "Register"
5. Check your email for verification link
6. Click the verification link
7. Return to app and log in

### Logging In
1. Enter your email and password
2. Tap "Login"
3. You'll be redirected to the home screen

## For Buyers

### Browsing Products

#### Home Screen
- View featured products and categories
- Scroll through product listings
- Tap on a product to view details

#### Search
1. Tap the search bar at the top
2. Enter product name or keywords
3. Results appear as you type
4. Tap on a product to view details

#### Categories
1. Scroll horizontally through category chips
2. Tap a category to filter products
3. Tap "All" to view all products

### Viewing Product Details
- Product images (swipe to view multiple)
- Product name and description
- Price (with discount if applicable)
- Stock availability
- Shop name (tap to visit shop)
- Customer reviews and ratings
- "Add to Cart" button

### Adding to Cart
1. On product details screen, tap "Add to Cart"
2. Product is added with quantity 1
3. View cart by tapping cart icon (top right)

### Managing Cart
1. Tap cart icon to view cart
2. **Increase quantity**: Tap "+" button
3. **Decrease quantity**: Tap "−" button
4. **Remove item**: Tap trash icon
5. **Clear cart**: Tap "Clear Cart" button
6. View total at bottom

### Checkout Process
1. In cart, tap "Checkout"
2. Enter or select shipping address:
   - Street address
   - City/Town
   - Province
   - Postal code
   - Contact number
3. Review order summary
4. Tap "Place Order"
5. Order is created and payment deducted from wallet

### Tracking Orders
1. Go to "Orders" tab
2. View all your orders
3. Tap an order to view details
4. Order statuses:
   - **Pending**: Waiting for seller approval
   - **Processing**: Seller is preparing order
   - **Approved**: Order confirmed by seller
   - **Ready for Delivery**: Ready to be delivered
   - **Completed**: Order delivered and confirmed
   - **Cancelled**: Order cancelled

### Receiving Orders
1. When order is "Ready for Delivery", seller will deliver
2. Upon receiving item, tap "Mark as Received"
3. Enter the delivery verification code provided by seller
4. Tap "Verify"
5. Order marked as completed
6. Funds released to seller

### Cancelling Orders
- **Full order**: Tap "Cancel Order" (within 12 hours)
- **Single item**: Tap "Cancel" on specific item (within 12 hours)
- Enter cancellation reason
- Refund credited to wallet

### Reviewing Products
1. Go to completed order
2. Tap "Leave Review" on product
3. Select star rating (1-5)
4. Write review text (optional)
5. Tap "Submit Review"
6. Review appears on product page

### Viewing Shop Status
- Shops can post 24-hour status updates (stories)
- Tap on shop status card to view
- Swipe to view multiple statuses
- Status disappears after 24 hours

## For Sellers

### Creating a Shop
1. Go to "My Shop" tab
2. Tap "Create Shop"
3. Enter shop details:
   - Shop name
   - Description
   - Location (town, province, country)
   - Contact number
   - Email (optional)
4. Tap "Create Shop"
5. Shop is created and you become a seller

### Managing Shop Profile
1. Go to "My Shop" tab
2. Tap "Edit Shop"
3. Update shop information
4. Tap "Save Changes"

### Adding Products
1. In "My Shop", tap "Add Product"
2. Enter product details:
   - Product name
   - Description
   - Price
   - Stock quantity
   - Category
3. Add product images (tap camera icon)
4. Tap "Create Product"

### Managing Products
1. View all your products in "My Shop"
2. Tap on a product to edit
3. Update details or images
4. Tap "Save Changes"
5. To delete: Tap "Delete Product"

### Setting Promotions
1. Edit a product
2. Scroll to "Promotion" section
3. Enter discount percentage
4. Set start and end dates
5. Tap "Save"
6. Discount automatically applied

### Managing Orders

#### Viewing Orders
1. Go to "Orders" tab in shop navigation
2. View "New Orders" and "Completed Orders"
3. Tap an order to view details

#### Processing Orders
1. **Approve Order**:
   - Tap "Approve Order"
   - Order status changes to "Processing"

2. **Mark Ready for Delivery**:
   - When order is packed, tap "Mark Ready"
   - Delivery code is generated
   - Share code with customer

3. **Confirm Delivery**:
   - Customer enters delivery code
   - Order automatically marked as completed
   - Funds transferred to your wallet

### Creating Status Updates
1. Go to "My Shop" tab
2. Tap "+" icon (create status)
3. Choose media type:
   - Text only
   - Image
   - Video
4. Add content
5. Select background color (for text)
6. Tap "Post Status"
7. Status visible for 24 hours

### Sharing Shop Link
1. Go to "My Shop" tab
2. Tap share icon (top right)
3. Choose sharing method
4. Link opens in web browser showing your shop

### Viewing Analytics
- Total products
- In-stock products
- Out-of-stock products
- New orders count
- Completed orders count

## Wallet Management

### Viewing Balance
1. Go to "Wallet" tab
2. View current balance at top
3. View recent transactions below

### Depositing Funds
1. In Wallet tab, tap "Deposit"
2. Enter amount
3. Choose payment method
4. Complete payment
5. Funds added to wallet

### Transaction History
1. Go to "Wallet" tab
2. Scroll to view all transactions
3. Transaction types:
   - **DEPOSIT**: Funds added
   - **PAYMENT**: Order payment
   - **REFUND**: Order cancellation refund
   - **PAYOUT**: Seller earnings

### Transaction Colors
- **Green**: Deposits and refunds (money in)
- **Red**: Payments (money out)
- **Blue**: Payouts to sellers

## Troubleshooting

### Can't Log In
- Verify email address is correct
- Check if email is verified (check inbox)
- Reset password if forgotten
- Ensure internet connection is active

### Product Not Adding to Cart
- Check if product is in stock
- Ensure you're logged in
- Try refreshing the app
- Clear app cache

### Order Not Appearing
- Pull down to refresh orders list
- Check internet connection
- Verify order was successfully placed
- Check email for order confirmation

### Delivery Code Not Working
- Ensure code is entered correctly (case-sensitive)
- Verify you're entering code for correct order
- Contact seller if code doesn't work
- Code expires after 7 days

### Wallet Balance Not Updating
- Pull down to refresh wallet
- Check transaction history
- Wait a few minutes for processing
- Contact support if issue persists

### Shop Not Visible
- Ensure shop is marked as active
- Check if shop has products
- Verify shop details are complete
- Contact admin if issue persists

### Images Not Loading
- Check internet connection
- Try refreshing the screen
- Clear app cache
- Ensure images were uploaded correctly

### App Crashes
- Update to latest version
- Clear app cache
- Restart device
- Reinstall app if necessary
